# RF-design-of-1.9-GHz-Rx-frontend
This project shows the design process of the main blocks of a typical RX frontend system:<br/>
1- Low-Noise Amplifier (LNA) <br/>
2- Local Oscillator (LO) <br/>
3- Mixer <br/>
